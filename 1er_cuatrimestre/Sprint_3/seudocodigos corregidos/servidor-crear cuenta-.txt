servidor -crear cuenta-

iniciar sistema

Recibe los datos de registro del cliente

Valida los datos en la base de datos

    Si los datos son inválidos o ya existen
        Envía mensaje al cliente: "Error en los datos" o "Correo ya registrado"
    Si los datos son válidos
        Guarda los datos en la base
        Genera un código de verificación
        Envía el código al correo del cliente
        Envía mensaje al cliente: "Código enviado, vence en 2 minutos"

Activa temporizador de expiración del código

Cuando el cliente envía el código:
    Si el código está expirado
        Envía mensaje: "Código expirado, se ha enviado uno nuevo"
        Genera nuevo código y lo envía
    Si el código es correcto y dentro del tiempo
        Marca la cuenta como verificada
        Envía mensaje: "Cuenta creada exitosamente"


Inicio del sistema

// Bloque 1: Preguntar si tiene cuenta
se pregunta al usuario si tiene una cuenta

// Bloque 2: Solicitar datos para crear cuenta
Si el usuario responde que no
    Se le manda a que cree una cuenta nueva ingresando:
        - Nombre
        - Correo electrónico
        - Contraseña
        - Teléfono
        - Fecha de nacimiento
        - sexo

    enviar los datos al servidor

// Bloque 3: Esperar validación inicial del servidor (datos válidos o no)
esperar validacion de los datos del servidor

si los datos devueltos son inválidos
    se le pedira que lo vuelva a intentar (volver a Bloque 2)

si los datos son válidos
    continuar al siguiente bloque

// Bloque 4: Esperar respuesta si la cuenta ya existe
esperar respuesta del servidor si existe la cuenta

si la cuenta ya existe
    se mostrará un mensaje indicando que esa cuenta ya existe
    se le pedirá crear con otros datos o iniciar sesión

si la cuenta no existe
    continuar al siguiente bloque

// Bloque 5: Esperar código de verificación
esperar respuesta del servidor con código de verificación

si se recibe el código de verificación
    mostrar mensaje para ingresar el código
    activar cronómetro de expiración

// Bloque 6: Validar código de verificación
Si el código es ingresado antes de que expire
    muestra un mensaje indicando que fue creada con éxito
    redirigir al inicio

si el código no es ingresado en tiempo
    mostrar un mensaje indicando que ya expiró
    solicitar nuevo código (volver a Bloque 5)

Fin del sistema
